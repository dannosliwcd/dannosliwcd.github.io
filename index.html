Daniel Wilson's Personal Page

Hi, I'm Daniel Wilson. I graduated with a PhD in Computer Engineering at Boston University in August 2023, advised by Prof. Ayse Coskun in the PeacLab. I have been researching opportunities to integrate job performance awareness into system-level power management policies in high-performance computing systems. As part of that work, I have been focusing on applications for quality-of-service-aware data center participation in demand response programs. During my PhD program, I have also been working as a research and software development intern on the GEOPM project at Intel. I am interested in continuing to explore ways we can make computers part of a more sustainable future.

This web page outlines some of my recent research and software contributions. Please check out my resume and reach out to me on LinkedIn or email me at danielcw@bu.edu.

Software Development

Open Software Contributions

  • Develop a software power management policy to GEOPM that configures non-uniform turbo frequency limits in a CPU to speed up imbalanced bulk-synchronous parallel applications.
  • Implement GEOPM interfaces to interact with PM Counters in Compute-Node Linux, and read or write settings for non-uniform turbo frequency limits.
  • Enhance GEOPM interfaces to interact with Model-Specific Registers (MSRs) by combining multiple read or write operations into a single io_uring batched operation.
  • Fix bugs, update documentation, and introduce new features in GEOPM since June 2019.
  • Fix bugs in the OpenStack unified SDK and implement missing management operations in the CLI client (Collected code reviews).
  • Fix a font-rendering bug in a PDF slide presentation tool.
  • Create a microbenchmark to explore relationships between performance and software power management decisions in imbalanced bulk-synchronous parallel applications.

Additional Software Experience

  • Develop tools and drivers that handle high-availability and disaster-recovery scenarios for software RAID in a clustered storage operating system (Data ONTAP).
  • Develop software to manage services and their allocated resources in an edge computing platform (Itron Riva)
